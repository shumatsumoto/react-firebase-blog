## :pencil2: Overview

Create a blog that integrates Firebase and React.

## :hammer: Requirement

- macOS
- React 17.0.1

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/react-firebase-blog.git
```
Move to directory
```
$ cd react-firebase-blog
```
Install the library
```
$ npm install
```
Launch a local server
```
$ npm run start
```
Access URL： 
http://localhost:3000
